# 4. Command-Line Help

Use these when you are not sure what a command does or which command to use.

## whatis

One-line description of a command:

```bash
whatis date
```

## man (manual pages)

Detailed documentation (options, examples, use cases):

```bash
man date
```

## -h / --help

Many commands support:

```bash
date -h
date --help
```

## apropos

Search man page names and descriptions by keyword:

```bash
apropos modpr
```

Use this to find commands related to a topic.
